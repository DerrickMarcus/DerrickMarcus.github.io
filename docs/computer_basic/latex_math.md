# LaTeX Math

本总结 $\LaTeX$ 中常用数学公式符号。主要参考 [通用 LaTeX 数学公式语法手册](http://www.uinio.com/Math/LaTex/)。

## 保留字符

$\LaTeX$ 中一些保留字符需要使用反斜杠 `\` 转义实现：

|   符号    |   LaTeX   |     符号     |    LaTeX     |
| :-------: | :-------: | :----------: | :----------: |
|   $\#$    |   `\#`    |     $\{$     |     `\{`     |
|   $\%$    |   `\%`    |     $\}$     |     `\}`     |
| $^\wedge$ | `^\wedge` |    $\sim$    |    `\sim`    |
|   $\&$    |   `\&`    | $\backslash$ | `\backslash` |
|   $\_$    |   `\_`    |              |              |
