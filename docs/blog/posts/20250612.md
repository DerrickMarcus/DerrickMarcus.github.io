# 2025年国铁第三季度调图

京哈高铁京沈段（北京至沈阳）、沪昆高铁杭长段（杭州至长沙）区间最高运营时速将提升至 350km/h。

贵南高铁 凤岭北线路所-永安镇 区间最高运营时速提升至 350km/h，部分途经此区间的，列车按 350km/h 标尺运行。

新线路开通：渝厦高铁渝黔段，最高运行时速 350km/h，区段达速（40.9km），拟安排直通列车17对.
